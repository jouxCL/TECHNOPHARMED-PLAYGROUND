======== INSTRUCCIONES PARA EL ADMINISTRADOR DEL HOSTING / BACKEND ========

Hola, 

Para que el formulario de contacto de la página web funcione correctamente, es necesario configurar las siguientes variables de entorno en el entorno de despliegue (como Cloudflare Pages o el servidor Node/Astro que se esté utilizando).

1. SERVICIOS REQUERIDOS:
- Google Cloud Platform: Para obtener credenciales (Service Account) con acceso a la API de Google Sheets.
- Google Sheets: La hoja de cálculo donde se insertarán los datos, la cual debe estar compartida con el correo del Service Account.
- Cloudflare Turnstile: Para el sistema Anti-Spam / Captcha del sitio web.

2. VARIABLES DE ENTORNO REQUERIDAS:
Por favor, asegúrate de configurar las siguientes variables en el entorno de producción. Los valores se obtendrán al configurar los servicios mencionados arriba:

GOOGLE_SHEET_ID="[El ID extraído de la URL de la hoja de Google Sheets. Ejemplo: 1XyZABCDEFG...]"

GOOGLE_SERVICE_ACCOUNT_EMAIL="[El correo electrónico 'client_email' del archivo JSON generado al crear la Service Account en Google Cloud. Ejemplo: tucuenta@tunombre.iam.gserviceaccount.com]"

GOOGLE_PRIVATE_KEY="[El 'private_key' del archivo JSON generado al crear la Service Account en Google Cloud. IMPORTANTE: Debe incluir todo el texto, incluyendo los bloques -----BEGIN PRIVATE KEY----- y -----END PRIVATE KEY-----]"

TURNSTILE_SECRET_KEY="[El 'Secret Key' privado generado en al añadir el dominio en Cloudflare Turnstile]"

PUBLIC_TURNSTILE_SITE_KEY="[El 'Site Key' público generado al añadir el dominio en Cloudflare Turnstile]"


Notas de implementación con la llave RSA privada:
Si utilizas un panel web (como Cloudflare Pages variables) para pegar el GOOGLE_PRIVATE_KEY, asegúrate de pegar la cadena con sus saltos de línea (puedes sustituir los \n literales si el hosting no los procesa automáticamente, aunque el código backend las estandariza).
