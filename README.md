# Technopharmed Playground

Sitio web para Technopharmed con formulario de contacto integrado con Google Sheets.

## 🚀 Características

- Formulario de contacto con validación
- Integración con Google Sheets API
- Diseño responsive
- Hosting en Cloudflare Pages

## 📦 Instalación

```bash
npm install
```

## 🛠 Desarrollo

```bash
npm run dev
```

## 🏗 Build

```bash
npm run build
```

## 🔧 Variables de Entorno

Crea un archivo `.env` basado en `.env.example`:

```env
GOOGLE_SHEET_ID=tu_id_de_google_sheet
GOOGLE_SERVICE_ACCOUNT_EMAIL=tu-service-account@project.iam.gserviceaccount.com
GOOGLE_PRIVATE_KEY=-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----\n
```
